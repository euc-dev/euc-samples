## Usage

Run the script, it will ask for Horizon Server FQDN, username, password and domain name to connect.  
The script will then present a menu:
- Login to Horizon
- List of Sessions
- List of Applications
- List of Machines
- Reboot a Desktop
- List of Desktop Pools
- Connection Server Info
- Usage Info
- Quit
